GAMMA EXTRACTION: FULL REPRODUCIBLE METHOD
For GPT to verify independently
============================================================

METHOD:

The gamma extraction has exactly TWO steps.

STEP 1: Fit a linear model to the PRE-THRESHOLD dispersions.
  - Use age bins 2-3, 3-4, 4-5, 5-6 Gyr (the flat era).
  - For each bin, compute sigma_U, sigma_V, sigma_W (standard deviation of 
    the U_UVW, V_UVW, W_UVW columns respectively).
  - Fit a straight line: sigma(t) = slope * t + intercept
  - This line represents the HEATING-ONLY baseline.

STEP 2: For each age bin (including post-threshold), compute:
  - gamma_X = observed_sigma_X / predicted_sigma_X
  - Where predicted_sigma_X comes from the linear extrapolation.

That's it. No fancy model. No free parameters beyond the linear fit.
The key assumption: the pre-threshold era (2-6 Gyr) represents pure
disc heating with no multiplicative signal (gamma = 1.0).

============================================================
PYTHON SCRIPT
============================================================

import pandas as pd
import numpy as np

# STEP 0: Load and filter
galah = pd.read_csv('galah_dr4_stars.csv')
galah_met = galah[(galah['fe_h'] >= -0.2) & (galah['fe_h'] <= 0.2) & (abs(galah['rv_comp_1']) < 150)]
solar = galah_met[(galah_met['mass'] >= 0.95) & (galah_met['mass'] < 1.05)]

# STEP 1: Compute raw dispersions for each 1-Gyr age bin
results = {}
for alo in range(1, 13):
    ahi = alo + 1
    ag = solar[(solar['age'] >= alo) & (solar['age'] < ahi)]
    if len(ag) >= 100:
        su = ag['U_UVW'].std()
        sv = ag['V_UVW'].std()
        sw = ag['W_UVW'].std()
        results[alo + 0.5] = (su, sv, sw, len(ag))

# Print raw dispersions
print(f"{'Age mid':>8} {'sigma_U':>10} {'sigma_V':>10} {'sigma_W':>10} {'n':>8}")
for t in sorted(results.keys()):
    su, sv, sw, n = results[t]
    print(f"{t:>8.1f} {su:>10.1f} {sv:>10.1f} {sw:>10.1f} {n:>8}")

# STEP 2: Linear fit to pre-threshold bins (2.5, 3.5, 4.5, 5.5)
pre_ages = np.array([2.5, 3.5, 4.5, 5.5])
pre_su = np.array([results[t][0] for t in pre_ages])
pre_sv = np.array([results[t][1] for t in pre_ages])
pre_sw = np.array([results[t][2] for t in pre_ages])

u_fit = np.polyfit(pre_ages, pre_su, 1)
v_fit = np.polyfit(pre_ages, pre_sv, 1)
w_fit = np.polyfit(pre_ages, pre_sw, 1)

print(f"\nLinear fits:")
print(f"  sigma_U = {u_fit[0]:.4f} * t + {u_fit[1]:.4f}")
print(f"  sigma_V = {v_fit[0]:.4f} * t + {v_fit[1]:.4f}")
print(f"  sigma_W = {w_fit[0]:.4f} * t + {w_fit[1]:.4f}")

# STEP 3: Compute gamma = observed / predicted
print(f"\n{'Age':>6} {'gam_U':>8} {'gam_V':>8} {'gam_W':>8} {'mean':>8} {'spread':>8}")
for t in sorted(results.keys()):
    su, sv, sw, n = results[t]
    pu = u_fit[0] * t + u_fit[1]
    pv = v_fit[0] * t + v_fit[1]
    pw = w_fit[0] * t + w_fit[1]
    gu = su / pu
    gv = sv / pv
    gw = sw / pw
    gm = np.mean([gu, gv, gw])
    spread = (max([gu, gv, gw]) - min([gu, gv, gw])) / gm * 100
    print(f"{t:>6.1f} {gu:>8.3f} {gv:>8.3f} {gw:>8.3f} {gm:>8.3f} {spread:>7.1f}%")
